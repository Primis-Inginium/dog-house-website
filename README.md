# 🐾 Primis Dog House Website

Standardized "Dog House" catalog and design suite for **Primis-Inginium**.

## 🏗 Modular Architecture
This project follows the **Core-Feature** modular pattern:
- `@core/*` (`src/core/`): Generic, reusable code (UI components, shared types).
- `@features/*` (`src/features/`): Domain-specific modules (Catalog, Design Suite).

## 🎨 Aesthetics
- **Vanilla CSS Modules**: High-performance, scoped styling.
- **Placeholder Strategy**: Stylized CSS-drawn dog houses instead of static assets.

## 🚀 Getting Started
1. `npm install`
2. `npm run dev`
